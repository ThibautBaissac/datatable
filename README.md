# DATATABLE

A simple yet dynamic datatable example using Rails 7.

Stack:
- Rails 7
- esbuild
- Hotwire
- Bootstrap 5
- Pagy
- pg_search

# Quick Start

1. Install Rails 7
  ```bash
  gem install rails
  ```

2. Clone
  ```bash
  git@github.com:ThibautBaissac/datatable.git
  ```

3. Bundle & yarn install
  ```bash
  bundle
  yarn install
  ```

4. Run rails db commands
  ```bash
  rails db:create
  rails db:migrate
  rails db:seed
  ```

5. Run rails server
  ```bash
  ./bin/dev
  ```
